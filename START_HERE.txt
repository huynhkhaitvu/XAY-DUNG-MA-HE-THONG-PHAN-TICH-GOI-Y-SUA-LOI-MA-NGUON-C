🔄 HƯỚNG DẪN CHI TIẾT
═══════════════════════════════════════════════════════════════

BƯỚC 1: Setup API (chạy 1 lần)
─────────────────────────────────
1. Nhấn đúp: SETUP_API_KEY.bat
2. Làm theo hướng dẫn trên màn hình
3. Kết quả: Tạo file backend\.env

BƯỚC 2: Cài Packages (chạy 1 lần)
──────────────────────────────────
1. Nhấn đúp: INSTALL_DEPENDENCIES.bat
2. Chờ cài xong (2-5 phút)
3. Bấm Enter để thoát

BƯỚC 3: Khởi Động (chạy mỗi lần)
──────────────────────────────────
1. Nhấn đúp: RUN.bat
2. Kết quả:
   • Terminal backend mở
   • Browser frontend mở
3. GIỮ TERMINAL BACKEND MỞ!

═══════════════════════════════════════════════════════════════
🆘 TROUBLESHOOT NHANH
═══════════════════════════════════════════════════════════════

❌ "GCC not found"
   → Cài MinGW-w64: https://www.mingw-w64.org/
   → Restart Windows

❌ "Python not found"
   → Cài Python: https://www.python.org/
   → Chọn "Add to PATH"
   → Restart Windows

❌ "Backend not responding"
   → Chạy: INSTALL_DEPENDENCIES.bat (cài lại)
   → Rồi: RUN.bat

❌ "Port 5000 occupied"
   → Đóng ứng dụng khác dùng port 5000
   → Hoặc restart Windows
