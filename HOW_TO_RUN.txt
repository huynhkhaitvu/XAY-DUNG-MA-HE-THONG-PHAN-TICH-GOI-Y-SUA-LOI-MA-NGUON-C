⚠️ QUAN TRỌNG
═══════════════════════════════════════

⛔ GIỮ TERMINAL BACKEND MỞ
   • Đừng đóng terminal (nếu bấm Ctrl+C)
   • Hệ thống sẽ không chạy

⛔ CHỰ ĐÓI BACKEND KHỞI ĐỘNG
   • Khi nhấn RUN.bat
   • Chờ 3-5 giây

⛔ FRESH BROWSER
   • Nếu lỗi kết nối
   • Bấm Ctrl+Shift+R (hard refresh)

═══════════════════════════════════════
🆘 TROUBLESHOOT
═══════════════════════════════════════

❌ "GCC not found"
   → Cài MinGW-w64
   → Thêm vào PATH
   → Restart Windows

❌ "Python not found"
   → Cài Python 3.8+
   → Chọn "Add to PATH"
   → Restart Windows

❌ "API Key error"
   → Kiểm tra backend\.env
   → Lấy key mới từ aistudio.google.com

❌ "Backend not responding"
   → Check terminal backend
   → Nếu có lỗi → fix + restart

❌ "Frontend không kết nối"
   → Refresh browser (Ctrl+R)
   → Check backend đang chạy
   → Check port 5000 không bị chiếm
